# Important Windows Event IDs

## Event ID 4624

Meaning:
Successful Logon

Why it matters?
Shows a user sucessfully logged into the system.

---

## Event ID 4625

Meaning:
Failed Logon

Why it matters?
May indicate password guessing or brute force attempts.

---

## Event ID 4672

Meaning:
Special Privileges Assigned

Why it matters?
Indicates administrative privileges were assigned during logon.

---

## Event ID 4688

Meaning:
Process Creation

Why it matters?
Shows what process was executed on the system.

Examples:

- powershell.exe
- cmd.exe
- mimikatz.exe

---

## Event ID 4720

Meaning:
User Account Created

Why it matters?
May indicate persistence or unauthorized account creation.

---

## Event ID 4728 /4732

Meaning:
User Added to a Privileged Group

Why it matters?
May indicate privilege escalation or unauthorized access.

## Key Learning
Event IDs help SOC analysts track logins, process execution, privilege escalation, and suspicious activity.
