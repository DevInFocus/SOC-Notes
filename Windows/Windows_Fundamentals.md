# Windows Fundamentals

## What is Windows?
Windows is an operating system used to manage hardware, software, users, files, and system resources.

## Why Windows matter in Cybersecurity?
Many organizations use Windows systems. SOC analysts frequently investigate Windows logs, user activity, processes, and security events.

## Users and Groups
Windows uses users and groups to control access and permissions.

Examples:

- Standard User
- Administrator
- Groups

## Windows Event Logs
Windows records system and security activity in Event Logs.

Important logs:

- Successful Logins
- Failed Logins
- Network Logins
- Administrative Logins

## Processes
A process is a running program on a system.

Examples:

- explorer.exe
- chrome.exe
- powershell.exe
- cmd.exe

## Key Learning
Windows logs are an important source of information during security investigations and incident response.
